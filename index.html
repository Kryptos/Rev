RevFramework

RevFramework is engineered with flexibiltity and scalability in mind.

The framework has a modular design based on the MVC architecture. There is true separation of framework and application, where you can have as many sites with as many themes running on the same instance of the framework.

RevFramework makes use of the Composer dependency-injection manager, so developers have as much freedom as possible.

User Interface

RevFramework does not come with a default UI. Nonetheless, since you're free to use any template engine you want, it's dead-easy to port your website over.
